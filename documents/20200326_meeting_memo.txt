Githubのファイルをためておく場所＝レポジトリ（リポジトリ）
GitHubの作業
gitというコマンドを使います．
Git for Windows というのをインストールしてください
作業のやり方：
1. git clone する　→　GitHubクラウド上のリポジトリをローカルにコピーする
2. git branch する　→　リポジトリの内容を変更するために自分用のブランチを作成する
3. git checkout する　→　作成したブランチに移動
4. ファイルを追加したり，更新したりする．
5. git add する　→　どの更新をカウントするかを宣言．
6. git commit する　→　更新を確定する．
7. git pushする　→　更新したローカルブランチを，クラウド上のリポジトリにアップロードする
8. GitHub上で，Pull Requestをする　→　変更をmasterブランチに統合してください，というリクエストを送る
9. 管理者がPull Requestをレビューして，変更がOKなら，mergeする．
